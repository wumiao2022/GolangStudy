这是一个简单的例子，展示如何使用Golang实现一个交互式命令行程序，用于读取用户输入和打印输出。

package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    reader := bufio.NewReader(os.Stdin)
    fmt.Print("Enter text: ")
    text, _ := reader.ReadString('\n')
    fmt.Println("You entered: ", text)
}

这个程序使用bufio包从标准输入读取一个文本行，并打印出来。
