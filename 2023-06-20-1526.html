我给你提供一个Golang的练习实例代码：求两个整数的最大公约数。

package main

import "fmt"

func main() {
    fmt.Println(gcd(60, 24)) // 12
    fmt.Println(gcd(36, 48)) // 12
    fmt.Println(gcd(17, 19)) // 1
}

func gcd(a, b int) int {
    if b == 0 {
        return a
    }
    return gcd(b, a%b)
}

这段代码中，我们定义了一个函数gcd，它的作用是求两个整数a和b的最大公约数。算法使用了辗转相除法，具体做法是对a和b进行取模，直到b等于0为止，此时a就是两个数的最大公约数。你可以使用这个例子来巩固你的算法能力和Golang编程能力。
