题目1：写一个函数，输入两个正整数n和m，输出它们的最大公约数。

func gcd(n, m int) int {
    if m == 0 {
        return n
    }
    return gcd(m, n%m)
}

题目2：写一个函数，输入一个字符串，判断它是否是回文字符串。

func isPalindrome(s string) bool {
    for i := 0; i < len(s)/2; i++ {
        if s[i] != s[len(s)-i-1] {
            return false
        }
    }
    return true
}

题目3：写一个函数，输入一个整数n，输出它的阶乘。

func factorial(n int) int {
    if n <= 1 {
        return 1
    }
    return n * factorial(n-1)
}
