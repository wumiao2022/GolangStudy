以下是一个示例练习代码，用于演示如何使用goroutine和channel实现一个简单的并发程序：

package main

import (
	"fmt"
	"time"
)

func main() {
	c := make(chan int)

	go func() {
		for i := 0; i < 5; i++ {
			c <- i
			time.Sleep(time.Second)
		}
	}()

	go func() {
		for {
			fmt.Println(<-c)
		}
	}()

	time.Sleep(time.Second * 6)
}

该程序会启动两个goroutine，一个从0到4循环发送数字到channel中，另一个无限循环从channel中读取数字并打印出来。运行该程序可以观察到两个goroutine会交替执行，实现了并发处理数据的功能。
