以下是一个Golang程序，可以从命令行接收一个字符串参数，并将其打印到控制台：

package main

import (
    "fmt"
    "os"
)

func main() {
    if len(os.Args) < 2 {
        fmt.Println("Missing argument: string to print")
    } else {
        fmt.Println(os.Args[1])
    }
}

我们可以在命令行中输入以下命令来运行该程序：

go run main.go "Hello, World!"

该程序将输出以下内容：

Hello, World!
