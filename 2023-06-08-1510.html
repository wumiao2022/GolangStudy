以下是一道练习题的代码，题目为：计算斐波那契数列的第n项（可以使用递归或迭代方式实现）：

递归方式实现：

func fibonacci(n int) int {
    if n <= 1 {
        return n
    }
    return fibonacci(n-1) + fibonacci(n-2)
}

迭代方式实现：

func fibonacci(n int) int {
    if n <= 1 {
        return n
    }
    var (
        a, b = 0, 1
        result int
    )
    for i := 2; i <= n; i++ {
        result = a + b
        a = b
        b = result
    }
    return result
}
