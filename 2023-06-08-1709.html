以下是一个简单的Golang练习题，实现一个函数，输入一个字符串，将其中的空格全部替换为”%20”。函数原型为：

func replaceSpace(str string) string {
    // 请在这里完成代码
}

代码如下：

func replaceSpace(str string) string {
    newStr := ""
    for _, r := range str {
        if r == ' ' {
            newStr += "%20"
        } else {
            newStr += string(r)
        }
    }
    return newStr
}

调用：

str := "hello world"
newStr := replaceSpace(str)
fmt.Println(newStr) // 打印结果为：hello%20world
