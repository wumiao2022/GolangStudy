这是一个简单的Golang程序，可以用来打印出Hello World的信息：

package main

import "fmt"

func main() {
    fmt.Println("Hello World!")
}

这是一个简单的计算两个整数之和并打印出结果的Golang程序：

package main

import "fmt"

func main() {
    var a, b = 10, 20
    var sum = a + b
    fmt.Println("Sum:", sum)
}

这是一个简单的从命令行读取用户输入的Golang程序：

package main

import (
    "bufio"
    "os"
    "fmt"
)

func main() {
    reader := bufio.NewReader(os.Stdin)

    fmt.Print("Enter Text: ")
    text, _ := reader.ReadString('\n')

    fmt.Println("You entered:", text)
}
