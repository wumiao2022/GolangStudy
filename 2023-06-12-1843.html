以下是Golang多线程的练习示例代码：

package main

import (
	"fmt"
	"sync"
)

var wg sync.WaitGroup

func main() {
	for i := 0; i < 10; i++ {
		wg.Add(1)
		go printNumber(i)
	}
	wg.Wait()
}

func printNumber(number int) {
	defer wg.Done()
	fmt.Println("Number:", number)
}

该示例代码创建了一个包含10个任务的多线程程序。每个任务都会调用一个名为printNumber的函数，该函数会打印相应的数字。使用sync包中的WaitGroup实现了主线程等待所有任务完成的功能。
