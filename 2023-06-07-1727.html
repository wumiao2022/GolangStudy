下面是一个简单的Golang练习示例，用于对数字进行排序。这个程序接收一个包含数字的切片，然后对其进行排序并返回排好序的切片。

func sortNumbers(numbers []int) []int {
    sort.Ints(numbers)
    return numbers
}

另一个示例程序是一个简单的Web服务器，监听8080端口并返回一个简单的字符串响应：

func main() {
    http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
        fmt.Fprintf(w, "Hello, World!")
    })

    http.ListenAndServe(":8080", nil)
}
