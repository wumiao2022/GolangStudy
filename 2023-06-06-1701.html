这是一个简单的Golang多 Goroutine 并发实现:

package main

import (
	"fmt"
	"sync"
)

func main() {
	var wg sync.WaitGroup
	for i := 1; i <= 10; i++ {
		wg.Add(1)
		go func(num int) {
			defer wg.Done()
			fmt.Printf("Goroutine %d\n", num)
		}(i)
	}
	wg.Wait()
	fmt.Println("All Goroutines are Done!")
}

这个程序将会启动 10 个 Goroutine 并行地执行，每个 Goroutine 简单地打印”Goroutine x”，最后打印”All Goroutines are Done!“。sync.WaitGroup 被用来等待所有的 Goroutine 完成执行。
