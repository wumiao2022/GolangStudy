以下是一个简单的Golang多线程程序，它创建了5个worker goroutine，并且每个goroutine都会打印一条消息：

package main

import (
	"fmt"
	"sync"
)

func main() {
	var wg sync.WaitGroup
	for i := 0; i < 5; i++ {
		wg.Add(1)
		go func(id int) {
			defer wg.Done()
			fmt.Printf("Worker %d started\n", id)
		}(i)
	}
	wg.Wait()
}

运行这个程序，你会看到下面的输出：

Worker 2 started
Worker 1 started
Worker 4 started
Worker 3 started
Worker 0 started

这个程序使用了sync包来等待所有的5个goroutine执行完毕，并且输出了它们的编号。你可以修改这个程序来实现不同的功能。
