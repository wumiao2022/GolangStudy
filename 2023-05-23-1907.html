这里是一份Golang练习实例代码，希望能帮助您提高编程能力：

  1. 使用for循环打印1到10的数字：
for i := 1; i <= 10; i++ {
    fmt.Println(i)
}
  1. 使用if语句判断一个数是否为偶数：
if num % 2 == 0 {
    fmt.Println("这是一个偶数")
} else {
    fmt.Println("这是一个奇数")
}
  1. 使用switch语句判断一个数的大小：
switch num {
case 1:
  fmt.Println("这是一个1")
case 2:
  fmt.Println("这是一个2")
case 3:
  fmt.Println("这是一个3")
default:
  fmt.Println("这是一个未知的数字")
}
  1. 使用数组进行排序操作：
var arr = []int{4, 2, 1, 3, 5}
sort.Ints(arr)
fmt.Println(arr)
  1. 使用切片进行遍历操作：
var slice = []string{"apple", "banana", "orange", "grape"}
for _, value := range slice {
    fmt.Println(value)
}
  1. 使用map存储键值对操作：
var m = make(map[string]int)
m["张三"] = 20
m["李四"] = 30
fmt.Println(m)
  1. 使用并发处理：
func main() {
    c := make(chan int)

    go func() {
        time.Sleep(time.Second)
        c <- 1
    }()

    fmt.Println(<-c)
}
