这是一个简单的Golang多线程样例，通过在主线程和子线程之间交替输出数字来演示多线程的基本用法。

package main

import (
	"fmt"
	"time"
)

func main() {
	go count()

	for i := 1; i <= 10; i++ {
		time.Sleep(1 * time.Second)
		fmt.Println("Main thread:", i)
	}
}

func count() {
	for i := 1; i <= 10; i++ {
		time.Sleep(2 * time.Second)
		fmt.Println("Child thread:", i)
	}
}

运行上述代码可以得到如下输出：

Main thread: 1
Child thread: 1
Child thread: 2
Main thread: 2
Child thread: 3
Child thread: 4
Main thread: 3
Child thread: 5
Child thread: 6
Main thread: 4
Child thread: 7
Child thread: 8
Main thread: 5
Child thread: 9
Child thread: 10
Main thread: 6
Main thread: 7
Main thread: 8
Main thread: 9
Main thread: 10

其中，“Main thread”和“Child thread”分别表示主线程和子线程。可以看到，它们交替输出数字，并且主线程总共输出了10个数字，而子线程只输出了10个数字。
