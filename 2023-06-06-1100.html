下面是一个简单的Golang练习实例，包括创建一个简单的HTTP服务器和处理HTTP请求：

package main
 
import (
    "fmt"
    "net/http"
)
 
func main() {
    http.HandleFunc("/", handler)
    http.ListenAndServe(":8080", nil)
}
 
func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello World!")
}

该程序创建一个HTTP服务器，监听端口8080，并在浏览器访问http://localhost:8080时返回字符串“Hello World!” 。
