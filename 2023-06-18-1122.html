以下是一个通过Golang并发实现的Hello World程序：

package main

import (
	"fmt"
	"sync"
)

func main() {
	var wg sync.WaitGroup
	wg.Add(2)

	go func() {
		defer wg.Done()
		fmt.Println("Hello")
	}()

	go func() {
		defer wg.Done()
		fmt.Println("World")
	}()

	wg.Wait()
}

这个程序会同时启动两个goroutine，每个goroutine负责打印一个单词。因为goroutine是并发执行的，所以两个单词会同时被输出。最后，主函数会通过WaitGroup等待所有goroutine执行完毕。
