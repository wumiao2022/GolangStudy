以下是一个基于Golang的HTTP服务器示例代码：

package main

import (
	"fmt"
	"net/http"
)

func main() {
	http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
		fmt.Fprintf(w, "Hello, world!")
	})

	http.ListenAndServe(":8080", nil)
}

该代码创建了一个HTTP服务器，并将根路径的请求响应输出为“Hello, world!”字符串。服务器监听端口号为8080。您可以通过运行代码并访问http://localhost:8080来测试它。
