可以在Go Playground（https://play.golang.org/）里练习以下例子：

  1. 计算1+2的结果并打印

    package main
import "fmt"
func main() {
fmt.Print(1 + 2)
}

  2. 定义一个变量，并将其赋值为字符串“Hello World”，然后将其打印出来

    package main
import "fmt"
func main() {
var str string = "Hello World"
fmt.Print(str)
}

  3. 定义一个函数，接受两个整数类型参数并返回它们的和

    package main
import "fmt"
func add(x int, y int) int {
return x + y
}
func main() {
fmt.Print(add(1, 2))
}

  4. 定义一个结构体类型Person，包含名字和年龄两个字段，然后创建一个Person类型的变量，并将其名字和年龄分别赋值为“张三”和18，并打印出来

    package main
import "fmt"
type Person struct {
Name string
Age int
}
func main() {
var p Person
p.Name = "张三"
p.Age = 18
fmt.Print(p.Name, ", ", p.Age)
}

  5. 定义一个切片类型，初始化该切片，使用append()函数添加元素，使用range关键字遍历切片并打印每个元素的值

    package main
import "fmt"
func main() {
var s []int
s = append(s, 1)
s = append(s, 2)
s = append(s, 3)
for _, num := range s {
    fmt.Print(num, " ")
}
}
