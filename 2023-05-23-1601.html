以下是一个简单的Golang练习实例，实现了对数组中元素的求和，将结果输出到控制台上：

package main

import "fmt"

func main() {
    nums := []int{1, 2, 3, 4, 5}
    sum := 0
    for _, num := range nums {
        sum += num
    }
    fmt.Println(sum)
}

这个实例展示了如何使用for循环和range关键字遍历一个切片类型的数组，并将每个元素累加到一个总和变量中。最后，使用fmt包的Println函数将计算出的总和输出到控制台上。

练习建议：修改数组中的元素并重新运行程序，看看输出的结果是否正确。尝试使用不同的for循环方式（如传统的for循环）来实现同样的功能。
