以下是一个使用Golang实现简单HTTP服务器的示例代码:

package main

import (
	"fmt"
	"net/http"
)

func main() {
	http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
		fmt.Fprintf(w, "Hello, World! This is a simple HTTP server written in Golang.")
	})

	http.ListenAndServe(":8080", nil)
}

这个程序会启动一个监听本地8080端口的HTTP服务器，对于任何URL的请求都会响应一个简单的”Hello, World!“消息。
