这是一个基于Golang的练习实例代码：

package main

import (
	"fmt"
	"math/rand"
	"time"
)

func main() {
	rand.Seed(time.Now().UnixNano()) // 设置随机种子

	// 生成一个长度为10，元素值为1到100之间的随机整数数组
	var arr [10]int
	for i := 0; i < 10; i++ {
		arr[i] = rand.Intn(100) + 1
	}

	// 输出数组
	fmt.Println("随机数组：", arr)

	// 计算数组元素的平均值
	var sum int
	for _, num := range arr {
		sum += num
	}
	avg := float64(sum) / float64(len(arr))

	// 输出平均值
	fmt.Printf("数组元素的平均值为：%.2f\n", avg)
}

这个程序实现了以下功能：

  1. 生成一个长度为10，元素值为1到100之间的随机整数数组。
  2. 计算数组元素的平均值。
  3. 输出数组和平均值。
