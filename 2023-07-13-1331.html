package main

import "fmt"

func main() {
	// 示例1
	var num1, num2 int
	fmt.Print("请输入两个整数：")
	fmt.Scan(&num1, &num2)
	fmt.Println("两个整数的和：", num1+num2)

	// 示例2
	var str1, str2 string
	fmt.Print("请输入两个字符串：")
	fmt.Scan(&str1, &str2)
	fmt.Println("拼接后的字符串：", str1+str2)

	// 示例3
	names := []string{"Alice", "Bob", "Charlie", "David"}
	for _, name := range names {
		fmt.Println("Hello, ", name)
	}
}

每个示例的功能如下：

示例1：从用户输入中读取两个整数，然后计算并输出它们的和。

示例2：从用户输入中读取两个字符串，并将它们拼接在一起后输出。

示例3：使用循环遍历字符串切片names，并输出每个字符串前面加上”Hello, “的结果。
