以下是一些Golang的练习题（附有代码解答）：

  1. 编写一个函数，接受一个字符串，返回这个字符串的第一个字母。
func firstLetter(str string) string {
    return string(str[0])
}
  1. 编写一个函数，接受一个整数，返回从1到该整数之间所有数字的和。
func sum(n int) int {
    sum := 0
    for i := 1; i <= n; i++ {
        sum += i
    }
    return sum
}
  1. 编写一个函数，接受两个字符串，返回它们拼接在一起的结果。
func concat(s1, s2 string) string {
    return s1 + s2
}
  1. 编写一个函数，接受一个整数数组，将数组中的元素按升序排列并返回新的数组。
func sortAsc(arr []int) []int {
    sort.Ints(arr)
    return arr
}
  1. 编写一个函数，接受一个字符串和一个字符，在字符串中找到该字符第一次出现的位置并返回索引值（如果字符串中没有该字符，返回-1）。
func indexOf(str string, c byte) int {
    for i := 0; i < len(str); i++ {
        if str[i] == c {
            return i
        }
    }
    return -1
}
