以下是一个简单的示例，演示如何使用Golang从标准输入中读取整数并计算其平均值：

package main

import "fmt"

func main() {
    var sum, count int
    for {
        var num int
        _, err := fmt.Scan(&num)
        if err != nil {
            break
        }
        sum += num
        count++
    }
    fmt.Printf("Average: %v\n", float64(sum)/float64(count))
}

这段代码使用fmt.Scan函数从标准输入中读取整数，并使用循环累加它们的总和和数量。最后，它打印出平均值。
