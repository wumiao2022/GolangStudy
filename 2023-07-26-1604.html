以下是一个多维数组的练习实例代码：

package main

import "fmt"

func main() {
    // 声明一个二维数组
    var matrix [3][3]int

    // 初始化二维数组的值
    matrix[0] = [3]int{1, 2, 3}
    matrix[1] = [3]int{4, 5, 6}
    matrix[2] = [3]int{7, 8, 9}

    // 打印二维数组的值
    for i := 0; i < len(matrix); i++ {
        for j := 0; j < len(matrix[i]); j++ {
            fmt.Printf("%d ", matrix[i][j])
        }
        fmt.Println()
    }
}

运行该代码，可以打印出以下输出结果：

1 2 3 
4 5 6 
7 8 9

这个练习帮助你熟悉了解如何声明和初始化多维数组，并通过嵌套的循环打印出数组中的值。继续多练习，加油！
