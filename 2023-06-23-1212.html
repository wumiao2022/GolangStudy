下面是一个基本的 HTTP 服务器：

package main

import (
	"fmt"
	"net/http"
)

func main() {
	http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
		fmt.Fprintf(w, "Hello World!")
	})

	http.ListenAndServe(":8080", nil)
}

这个程序创建了一个基础的 HTTP 服务器，当用户访问网站时，将返回”Hello World!“这个字符串。该程序还监听了端口8080来处理来自客户端的请求。
