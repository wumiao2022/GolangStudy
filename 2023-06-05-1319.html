这是一道简单的Golang练习题，实现一个函数，输入一个整数n，返回从1-n的所有整数的和。

func sum(n int) int {
    s := 0
    for i := 1; i <= n; i++ {
        s += i
    }
    return s
}

这是一道中等难度的Golang 练习题，实现一个函数，输入n个数字（可以动态输入），输出其中的最大值。

func max(nums ...int) int {
    if len(nums) == 0 {
        panic("空切片")
    }
    max := nums[0]
    for _, num := range nums {
        if num > max {
            max = num
        }
    }
    return max
}

这是一道高难度的Golang 练习题，实现一个函数，输入一个字符串，判断该字符串是否是回文串（即正着读和倒着读都是一样的）。

func isPalindrome(s string) bool {
    n := len(s)
    for i := 0; i < n/2; i++ {
        if s[i] != s[n-1-i] {
            return false
        }
    }
    return true
}
