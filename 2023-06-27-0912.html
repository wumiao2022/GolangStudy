以下是一个简单的Golang练习：

package main

import "fmt"

func main() {
	// 验证数组是值类型
	array1 := [3]int{1, 2, 3}
	array2 := array1
	array2[0] = 0
	fmt.Println(array1) // 输出 [1 2 3]
	fmt.Println(array2) // 输出 [0 2 3]

	// 验证切片是引用类型
	slice1 := []int{1, 2, 3}
	slice2 := slice1
	slice2[0] = 0
	fmt.Println(slice1) // 输出 [0 2 3]
	fmt.Println(slice2) // 输出 [0 2 3]

	// 验证map是引用类型
	map1 := map[string]string{"name": "Alice", "age": "20"}
	map2 := map1
	map2["name"] = "Bob"
	fmt.Println(map1) // 输出 map[age:20 name:Bob]
	fmt.Println(map2) // 输出 map[age:20 name:Bob]
}

这个练习主要是帮助您理解Golang中的值类型和引用类型。在验证中，我们创建了一个数组，一个切片和一个映射，然后将它们分别赋值给另外一个变量。我们更改第二个变量的值，并打印两个变量，最终输出结果会帮助我们了解到值类型和引用类型的区别。
