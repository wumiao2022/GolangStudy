以下是一段Golang多协程并发执行的代码：

package main

import (
    "fmt"
    "sync"
)

func main() {
    var wg sync.WaitGroup

    for i := 1; i <= 10; i++ {
        wg.Add(1)
        go func() {
            fmt.Printf("Hello from goroutine %v\n", i)
            wg.Done()
        }()
    }

    wg.Wait()
}

以上代码创建了10个协程，并发执行输出这10个协程执行的序号。使用了WaitGroup来等待所有协程执行完毕后再结束程序。
